## ⚙️ Setup & Run Locally

```bash
# Clone the repository
git clone https://github.com/kozmarti/portfolio.git

# Navigate into the folder
cd portfolio

# Install dependencies
npm install

# Run development server
npm run dev

```
Then open http://localhost:3000 to view it in your browser.
